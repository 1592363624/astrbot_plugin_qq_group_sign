# QQ群打卡插件

一个功能强大的QQ群打卡插件，支持自动定时打卡、白名单模式、管理员通知等功能。适配 AstrBot v4.25+。

> 最新版本：`v2.3.0`

## 主要功能

- 自动定时打卡：支持自定义时分秒的打卡时间
- 白名单模式：可以指定特定群聊进行打卡
- 全群模式：默认打卡所有群聊
- 统计功能：记录打卡成功/失败次数
- 管理员通知：打卡完成后通知管理员或管理群
- 手动打卡：支持在当前群聊或全群执行打卡
- 指令菜单：提供完整的指令列表和使用说明

## 安装说明

1. 将插件文件夹 `astrbot_plugin_qq_group_sign` 放入 AstrBot 的 `data/plugins` 目录
2. 在 AstrBot WebUI 中启用插件
3. 根据需要配置插件参数

## 配置说明

- **打卡时间**：格式 `HH:MM:SS`，例如 `08:00:00`
- **时区设置**：东八区为 `8`
- **启用自动打卡**：是否开启自动定时打卡
- **白名单模式**：`true`=只打卡白名单群聊，`false`=打卡所有群聊
- **管理员通知**：打卡完成后是否通知管理员
- **管理群号**：接收打卡通知的管理群号
- **打卡消息**：打卡时发送的消息内容

## 使用命令

### 基础打卡命令
| 命令 | 别名 | 说明 |
|------|------|------|
| `/打卡` | `/群打卡` | 在当前群聊执行打卡 |
| `/全群打卡` | `/打卡所有群` | 对所有群聊执行打卡 |
| `/打卡菜单` | - | 显示所有可用指令 |

### 白名单管理
| 命令 | 别名 | 说明 |
|------|------|------|
| `/添加白名单` | `/加白名单` | 添加群号到白名单 |
| `/移除白名单` | `/删白名单` | 从白名单移除群号 |
| `/查看白名单` | `/白名单列表` | 查看白名单列表 |

### 自动打卡控制
| 命令 | 别名 | 说明 |
|------|------|------|
| `/开启自动打卡` | `/启动打卡` | 开启自动打卡 |
| `/关闭自动打卡` | `/停止打卡` | 关闭自动打卡 |
| `/设置打卡时间` | `/打卡时间` | 设置打卡时间 |
| `/切换模式` | `/打卡模式` | 切换白名单/全群模式 |

### 状态查询
| 命令 | 别名 | 说明 |
|------|------|------|
| `/打卡状态` | `/打卡统计` | 查看打卡状态和统计 |

## 更新日志

### v2.3.0
- 适配 AstrBot v4.25+ MessageType 变更（session 字符串使用 `GroupMessage` 替代旧 `group`）
- 修复裸 `except` 捕获，改为指定异常类型（`ValueError/TypeError/OSError`）
- `全群打卡` 命令改用 `@filter.regex` 匹配，避免前缀干扰导致不触发
- 统计结果只回发到命令发起的原会话，不在目标群发文本
- 版本号对齐（metadata 与 @register 一致）

### v2.2.0
- 改用 `send_group_sign` API 直接签到（不再发文本冒充打卡）
- 全群打卡统计汇总只发到命令来源会话
- 支持 LLBot / NapCat 双协议

### v2.1.0
- 初始公开版本

## 注意事项

1. 确保机器人有在目标群聊的权限
2. 打卡时间必须使用 `HH:MM:SS` 格式
3. 需要 AstrBot >= 4.16
